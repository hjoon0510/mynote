
# How to backup source code to github.com webstie

## download files
```
git clone <br>
https://github.com/hjoon0510/test.git <br>
cd <folder-name> <br>
```
## edit 
```
gedit < file-name >   
cat  < file-name >  
ls   
```


## backup (to github.com)
```
git add <file-name>
git commit -a -s
git pull
git push origin master
-your id: ****
-your password: ****
```
